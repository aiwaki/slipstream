Slipstream — приложение в строке меню macOS. Оно незаметно чинит то, что
провайдер вытворяет с твоим соединением: замедляет, рвёт, подсовывает
фальшивый DNS, режет по гео. И при этом **не гонит весь твой трафик через VPN.**
Поставил, вошёл в свой Geph — и забыл, что оно есть. Ни расширений, ни возни с
прокси в каждой программе, ни кнопок «вкл/выкл» в обычной жизни.

Заточено под российский TSPU, но приём сработает против любой DPI-цензуры.

> [!NOTE]
> **Пока рано:** ранняя версия. macOS на Apple Silicon, настроено под сети РФ.

## Что чинит

- **Discord** — чат, серверы, картинки и **голос** (тот самый UDP, который почти все инструменты роняют).
- **YouTube** — без подвисаний от замедления.
- **ChatGPT и Claude** — сервисы, которые в лоб не пускают российские IP, идут через реальный выход за границей и просто работают.
- **Telegram** — Desktop ходит через встроенный локальный прокси мимо блокировки по IP дата-центров; включается одной кнопкой.
- **Почти всё заблокированное, замедленное или отравленное по DNS** — само собой.

## Почему ощущается иначе

**Это не обычный VPN, который заворачивает вообще всё.** Slipstream делит трафик
специально:

- Российское (банки, госуслуги, местные сайты) идёт **напрямую и быстро** — оно вообще не уезжает из страны.
- То, что провайдер просто *ломает* (замедление, обрыв, фальшивый DNS), чинится **на месте, локально** — твой IP остаётся российским, отваливается только сама блокировка.
- И лишь те немногие сервисы, что **отказывают российским IP целиком**, уходят в зашифрованный обход за границу.

Быстро там, где можно; скрытно там, где надо; и тихо всё время.

## Как это работает

Три отдельных пути, которые никогда не смешиваются:

```
                 ┌─────────────────────────── твой Mac ───────────────────────────┐
   любая         │  прозрачный перехват :443 (pf)                                  │
   программа ──► │        │                                                        │
   (браузер,     │        ├─ российский хост? → напрямую, не трогаем               │
   Discord,      │        ├─ ломают по DPI?   → 1) ДЕСИНК (локально, на месте)     │
   Claude…)      │        └─ режут по гео?    → 2) ТУННЕЛЬ GEPH (выход за границей) │
                 │  Telegram Desktop ───────► 3) TG-WS-ПРОКСИ (локальный MTProto)   │
                 └────────────────────────────────────────────────────────────────┘
```

1. **Десинк** *(локально, без сервера)* — обманывает DPI-фильтр: рвёт TLS-запись
   на части и подсовывает пакеты-обманки с низким TTL (по мотивам zapret /
   byedpi), плюс DoH против подмены DNS и отдельная UDP-дорожка для голоса
   Discord. Твой IP не меняется вообще.
2. **Туннель Geph** — открытая сеть [Geph](https://geph.io) (`geph5-client`
   внутри) выводит за границу **только** то, что режут по гео, через выбранную
   тобой страну. Остальное туннель не трогает.
3. **Telegram-прокси** — [tg-ws-proxy](https://github.com/Flowseal/tg-ws-proxy)
   внутри поднимает локальный MTProto-прокси, который гонит Telegram по
   WebSocket мимо блокировки IP дата-центров.

Ничего не крутится на наших серверах — их просто нет. Geph — твой аккаунт в уже
готовой открытой сети; десинк и Telegram-прокси целиком у тебя на машине.

## Установка

1. Скачай `Slipstream.app` из [Releases](https://github.com/aiwaki/slipstream/releases) и перетащи в «Программы».
2. Запусти. При первом старте оно **один раз спросит пароль** — чтобы поставить фоновую службу. Дальше всё делает само.
3. Открой меню в строке меню:
   - **Geph → Account…** — вставь ключ своего аккаунта Geph (хватит бесплатного).
   - **Geph → выбери выход** — город или **Automatic**.
   - **Connect Telegram Proxy** — направит Telegram Desktop на встроенный прокси.

Всё. Дальше оно не отсвечивает. Больше руками ничего делать не нужно.

## Сборка из исходников

Нужны Rust, Node, Python 3 и Xcode command-line tools.

```bash
# приложение в строке меню
cd app-tauri && npm install && npm run tauri build

# фоновая служба (десинк + роутинг) — приложение ставит её само,
# но при разработке можно поставить вручную:
sudo python3 spike/tproxy.py --install
```

Встроенный `geph5-client` собирается из исходников прямо в CI
([`build-geph.yml`](.github/workflows/build-geph.yml)) — всегда свежий, никаких
протухших бинарников.

## Что где лежит

| Путь | Что это |
|------|---------|
| `app-tauri/` | Приложение в строке меню — нативный интерфейс macOS (Tauri + Rust). |
| `spike/tproxy.py` | Служба прозрачного десинка и раздельного роутинга (Python, root). |
| `vendor/tg-ws-proxy/` | Встроенный Telegram-прокси MTProto-over-WebSocket. |
| `vendor/geph/` | Как собирается и обновляется встроенный `geph5-client`. |
| `docs/` | Заметки по устройству и модель угроз. |

## Приватность и доверие

- Всё крутится на твоей машине; своего бэкенда у Slipstream нет.
- Раздельный трафик держит российские сервисы **в стороне** от любого туннеля — они идут напрямую, и твой банк никогда не видит заграничный IP.
- Geph — твой собственный аккаунт в открытой сети Geph; их модель угроз — в их документации.
